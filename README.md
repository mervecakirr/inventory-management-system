# 📦 Inventory Management System

> Enterprise Inventory Management System developed with **Java Spring Boot**, following **SOLID principles** and implementing **Gang of Four (GoF) Design Patterns**.

---

## 🎯 Project Goal

This project demonstrates how to build a scalable Inventory Management System using modern software engineering practices.

The project is intended for:

- Software Engineering students
- Java developers
- Spring Boot learners
- Design Pattern enthusiasts
- GitHub portfolio

---

## 🚀 Features

- Product Management
- Warehouse Management
- Supplier Management
- Customer Management
- Purchase Orders
- Sales Orders
- Stock Tracking
- Inventory Reports
- Notification System
- ERP Integration
- Role-Based Authorization
